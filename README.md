# Student Grading System

## Description
Desktop-based application built with Java Swing to manage student records, grades, and marks.

## Features
- Add, update, and delete student records
- Enter marks and calculate grades automatically
- User-friendly interface

## Technologies Used
- Java
- Java Swing

## How to Run
1. Clone the repository
2. Open the project in NetBeans or IntelliJ IDEA
3. Run Main.java
